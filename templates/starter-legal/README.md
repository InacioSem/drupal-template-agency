# DrupalReady Starter Legal Template

Production-ready Drupal 11 template. Part of [DrupalReady](https://drupalready.dev).

## Quick Start

```bash
composer create-project drupalready/starter-legal my-site
cd my-site
lando start
lando drush site:install --existing-config -y
```

## Documentation

Visit [drupalready.dev/docs/getting-started](https://drupalready.dev/docs/getting-started) for full setup instructions.
